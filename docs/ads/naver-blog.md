# Naver Blog — AFH Permit Automation

## 제목 (Title)

워싱턴 AFH(성인가족홈) 허가 준비 — 건축가 없이 도면·체크리스트 자동화

---

## 본문 (Body) — Korean

**서론**  
워싱턴주에서 단독주택을 성인가족홈(AFH)으로 전환할 때, 허가 서류(도면, WAC/IRC 준수, 건축 검사 체크리스트)가 부담될 수 있습니다. 건축가 없이 대부분을 직접 준비할 수 있는 웹 앱을 소개합니다.

**기능**  
- **부동산·평면도 입력** — 주소, 도시, 우편번호, 연면적, 각 실(침실, 욕실, 거실, 주방)의 치수 입력  
- **WAC/IRC/WABO 준수** — 침실 70 sq ft 이상, 비상창 5.7 sq ft 이상, 32" 문, 그랩바, 경사로, 연기/CO 경보, 소화기. 앱이 검사관 요구사항에 맞는지 점검  
- **허가용 텍스트 도면** — 복사해서 신청서에 첨부하거나, isometric → mixboard → notebooklm 워크플로로 3D/제안서 PDF 제작  
- **허가 양식·사례** — Centralia, Lewis County, WABO AFH 체크리스트 + 2606 Cooks Hill Rd 사례(현재 vs 변경 후 도면·제출 패키지)  
- **컨설턴트 교육** — 8주 커리큘럼, AFH 관리 Top 100 문항, Moltbot 스크립트로 서류 자동화  

**대상**  
워싱턴 AFH 운영자·컨설턴트(Centralia, Lewis County, Chehalis, Lacey, Tumwater, Olympia 등). DIY 허가 준비용.
